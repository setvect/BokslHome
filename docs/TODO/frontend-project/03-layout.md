# 3단계: 기본 레이아웃 (Simple Layout)

## 목표
- 단순하고 안정적인 반응형 레이아웃 구조 구축
- 기본 헤더 + 메인 콘텐츠 구조 구현
- 복잡한 전역 상태 관리 피하고 단순한 로컬 상태부터 시작
- 접근성을 고려한 네비게이션 구현
